Iniciando tutorial de Python

Oque e Python

Python e uma linguagem de programação de alto nivel, interpretada e dinamica. É também conhecida por
ter sua sintaxe clara e simples, oque facilita para que novos programadores utilizem ela por ser uma
otima escolha para iniciantes.

Passo a passo downloads

Primeiro passo para fazer o dowload do python, voce tem que assessar o site oficial do Python em 
https://www.python.org/downloads/

Segundo passo agora ja dentro do site procure a versão mais recente antes de fazer o dowload.
Hoje dia 23/04/2024 a versão mais recente e a Python 3.11.2.

Terceiro passo click no link do instalador executavel de acordo com as bits do deu sistema podendo ser
Windows x86-64 ou Windows x86-32

Passo a passo da Instalação do python

Primeiro passo localize o arquivo do instalador do Python que você baixou e execute-o

Segundo passo na primeira tela do instalador, selecione a opção Instalação personalizada

Terceiro passo na próxima tela, selecione a pasta de instalação do Python. Recomendo deixar a opção 
padrão.

Quarto passo na próxima tela, você verá uma lista de componentes do Python que podem ser instalados.
recomendo selecionar todas as opções, exceto Documentação offline e Python Test Suite.

Quinto passo na próxima tela, Selecione essa opção Add Python 3.x to PATH, so facilitará a execução 
de arquivos Python a partir da linha de comando.

Sexto passo na próxima tela, você verá uma resumo das opções de instalação selecionadas. Se tudo estiver 
correto, clique no botão Install para iniciar a instalação.

setimo passo Após a conclusão da instalação, você pode verificar se o Python foi instalado corretamente 
abrindo uma linha de comando e digitando o seguinte comando:
python --version
Se o Python foi instalado corretamente, você verá a versão do Python que você instalou exibida na linha
de comando.
